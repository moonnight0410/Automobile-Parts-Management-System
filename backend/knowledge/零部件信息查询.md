# 零部件信息查询

## 概述

零部件信息查询是了解零部件基本信息、生产信息、质保信息和流通信息的重要功能。本系统通过区块链技术记录零部件全生命周期信息，确保信息的真实性和可追溯性。

## 查询方式

### 方式1：通过零部件ID查询
1. 进入"零部件管理 > 零部件信息"页面
2. 在搜索框中输入零部件ID（如：ENG-PISTON-202505）
3. 点击"查询"按钮
4. 系统会显示该零部件的详细信息

### 方式2：通过批次号查询
1. 进入"零部件管理 > 零部件信息"页面
2. 在搜索框中输入批次号（如：BATCH-2024-001）
3. 点击"查询"按钮
4. 系统会显示该批次所有零部件的信息

### 方式3：通过二维码查询
1. 使用手机扫描零部件上的二维码
2. 系统自动识别零部件ID
3. 显示零部件详细信息

### 方式4：通过AI助手查询
1. 进入"售后管理 > 智能售后助手"页面
2. 输入问题，如："零部件ENG-PISTON-202505的信息"
3. AI助手会识别意图并查询区块链数据
4. 返回零部件详细信息

## 零部件信息内容

### 基本信息
- **零部件ID**：唯一标识符（格式：类型-名称-年月）
- **零部件名称**：零部件的正式名称
- **零部件类型**：发动机、变速箱、制动系统等
- **零部件型号**：零部件的具体型号
- **规格参数**：零部件的技术参数
- **制造商**：零部件的生产厂家
- **生产日期**：零部件的生产日期
- **批次号**：零部件的批次编号

### 质保信息
- **质保状态**：在质保期内/已过质保期
- **质保起算时间**：质保期开始的时间
- **质保到期时间**：质保期结束的时间
- **质保范围**：质保覆盖的内容
- **质保类型**：标准质保/延保

### 流通信息
- **生产记录**：生产时间、生产地点、生产人员
- **入库记录**：入库时间、入库仓库、入库人员
- **出库记录**：出库时间、出库仓库、出库人员
- **运输记录**：运输时间、运输方式、运输公司
- **安装记录**：安装时间、安装地点、安装人员
- **维修记录**：维修时间、维修内容、维修人员

### 状态信息
- **当前状态**：在库/在途/已安装/已维修/已报废
- **当前位置**：零部件当前所在位置
- **使用情况**：零部件使用状态和性能
- **健康状态**：零部件健康度评估

## 常见问题

### Q1：如何获取零部件ID？

A：零部件ID获取方式：
1. **扫描二维码**：扫描零部件上的二维码
2. **查看标签**：查看零部件包装上的标签
3. **联系供应商**：提供零部件名称和型号，供应商会提供零部件ID
4. **查看维修记录**：查看之前的维修记录
5. **到授权机构检测**：到授权维修中心，技术人员会检测并识别

### Q2：零部件ID格式是什么？

A：零部件ID格式：
- **格式**：类型-名称-年月
- **示例**：ENG-PISTON-202505
- **说明**：
  - ENG：零部件类型（发动机）
  - PISTON：零部件名称（活塞）
  - 202505：生产年月（2025年5月）

### Q3：如何查询零部件是否在质保期内？

A：可以通过以下方式查询：
1. **通过零部件ID查询**：输入零部件ID，系统显示质保状态
2. **通过VIN码查询**：输入车辆VIN码，系统显示整车质保信息
3. **到授权机构查询**：到授权维修中心查询
4. **联系客服查询**：联系客服协助查询

### Q4：零部件信息可以修改吗？

A：零部件信息一旦上链，将永久保存，无法修改。原因：
1. 确保数据的真实性和可追溯性
2. 满足监管要求
3. 防止恶意篡改记录

### Q5：零部件信息可以分享吗？

1. **截图分享**：截图分享零部件信息页面
2. **提供零部件ID**：提供零部件ID让对方自行查询
3. **联系客服**：联系客服，授权他人查询

### Q6：零部件信息查询收费吗？

A：零部件信息查询完全免费。包括：
1. 基本信息查询
2. 质保信息查询
3. 流通信息查询
4. 不收取任何其他费用

### Q7：零部件信息准确吗？

A：是的。零部件信息准确可靠：
1. 所有信息上链存证
2. 确保数据真实可追溯
3. 无法篡改或删除
4. 满足监管要求

### Q8：零部件信息多久更新一次？

A：零部件信息实时更新：
1. 生产、入库、出库、运输、安装、维修等操作都会实时上链
2. 系统会自动更新零部件状态
3. 用户可以实时查询最新信息

### Q9：零部件信息可以导出吗？

A：可以。您可以：
1. **导出PDF**：导出零部件信息为PDF文件
2. **导出Excel**：导出零部件信息为Excel文件
3. **打印**：打印零部件信息
