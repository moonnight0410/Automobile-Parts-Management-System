# 召回信息查询

## 概述

零部件召回是保障车辆安全的重要措施。用户可以通过批次号、零部件ID或VIN码查询零部件是否在召回范围内。召回信息会上链存证，确保透明可追溯。

## 查询方式

### 方式1：通过批次号查询

1. 进入"售后管理 > 召回记录"页面
2. 在搜索框中输入零部件批次号
3. 点击"查询"按钮
4. 系统会显示该批次的召回信息

### 方式2：通过零部件ID查询

1. 进入"售后管理 > 召回记录"页面
2. 在搜索框中输入零部件ID
3. 点击"查询"按钮
4. 系统会显示该零部件的召回信息

### 方式3：通过VIN码查询

1. 进入"售后管理 > 召回记录"页面
2. 在搜索框中输入车辆VIN码
3. 点击"查询"按钮
4. 系统会显示该车辆涉及的所有召回信息

### 方式4：通过AI助手查询

1. 进入"售后管理 > 智能售后助手"页面
2. 输入问题，如："批次BATCH-2024-001在召回范围内吗？"
3. AI助手会识别意图并查询区块链数据
4. 返回召回状态和详细信息

## 召回状态说明

### 1. 正常（Normal）
- **含义**：零部件不在召回范围内
- **用户操作**：无需操作
- **建议**：正常使用，定期检查

### 2. 召回中（Recalling）
- **含义**：零部件已被召回，正在执行召回措施
- **召回原因**：存在安全隐患或质量缺陷
- **用户操作**：
  - 尽快联系售后中心
  - 安排免费检查或更换
  - 配合召回流程

### 3. 待召回（Pending Recall）
- **含义**：零部件已确定需要召回，但召回尚未开始
- **召回原因**：已识别的问题
- **召回时间**：计划召回的开始日期
- **用户操作**：
  - 关注召回通知
  - 提前预约售后服务
  - 准备相关材料

### 4. 已完成（Completed）
- **含义**：召回措施已执行完成
- **召回时间**：实际执行的日期
- **用户操作**：
  - 确认更换效果
  - 保留更换记录
  - 注意后续维护

### 5. 已取消（Cancelled）
- **含义**：召回计划已取消
- **取消原因**：
  - 问题已通过其他方式解决
  - 风险评估结果更新
  - 其他技术原因
- **用户操作**：无需操作

## 召回信息详情

### 基本信息
- **召回编号**：唯一标识召回事件（格式：RECALL-YYYYMMDD-XXX）
- **召回批次**：受影响的零部件批次号
- **零部件名称**：被召回的零部件名称
- **零部件ID**：零部件的唯一标识符

### 召回原因
- **问题描述**：详细说明存在的安全隐患或质量缺陷
- **风险等级**：
  - 高风险：存在严重安全隐患
  - 中风险：存在安全隐患
  - 低风险：存在质量缺陷
- **影响范围**：受影响的车辆数量和地区

### 召回措施
- **处理方式**：
  - 免费更换：更换新零部件
  - 免费维修：修复缺陷
  - 检查确认：检查是否受影响
- **执行时间**：召回开始和结束时间
- **负责机构**：执行召回的授权机构

### 用户操作指引
- **操作步骤**：详细的召回执行流程
- **所需材料**：身份证、行驶证、车辆登记证等
- **预约方式**：电话预约、在线预约、到店预约
- **注意事项**：召回过程中的注意事项

## 召回流程

### 1. 召回通知
- **通知方式**：
  - 短信通知
  - 邮件通知
  - 系统消息
  - 媒体公告
- **通知内容**：
  - 召回编号
  - 受影响零部件
  - 召回原因
  - 处理方式

### 2. 用户确认
- **确认方式**：
  - 在线确认收到通知
  - 到店确认
  - 电话确认
- **确认内容**：
  - 确认车辆信息
  - 确认零部件信息
  - 选择预约时间

### 3. 预约服务
- **预约渠道**：
  - 在线预约系统
  - 客服电话预约
  - 到店预约
- **预约信息**：
  - 车辆信息
  - 预约时间
  - 联系方式

### 4. 执行召回
- **执行内容**：
  - 检查零部件
  - 更换新零部件
  - 维修处理
- **执行时间**：根据召回措施而定
- **费用**：完全免费

### 5. 记录存证
- **记录内容**：
  - 召回执行记录
  - 更换零部件信息
  - 用户确认签字
- **上链存证**：所有记录上链，确保可追溯

## 常见问题

### Q1：召回是免费的吗？

A：是的，召回完全免费。包括：
1. 零部件更换费用
2. 维修工时费用
3. 检查费用
4. 不收取任何其他费用

### Q2：召回会影响质保吗？

A：不会。召回是独立的处理措施：
1. 不影响原质保期
2. 更换的零部件享有独立质保
3. 召回处理不计入维修次数

### Q3：召回必须执行吗？

A：强烈建议执行。召回零部件存在安全隐患：
1. 继续使用可能危及安全
2. 发生事故可能影响质保
3. 法律法规要求配合召回

### Q4：召回需要多长时间？

A：召回处理时间：
- 检查确认：30分钟-1小时
- 更换零部件：1-2小时
- 维修处理：2-4小时
- 具体时间根据召回措施而定

### Q5：召回可以跨地区执行吗？

A：可以。召回可以在任何授权机构执行：
1. 不受地区限制
2. 全国授权机构均可执行
3. 建议选择就近的机构

### Q6：召回后零部件质保期如何计算？

A：召回更换的零部件：
1. 享有独立质保期
2. 质保期从更换之日起计算
3. 质保期与原零部件相同

### Q7：如何知道我的零部件是否在召回范围内？

A：可以通过以下方式查询：
1. 输入零部件批次号查询
2. 输入零部件ID查询
3. 输入VIN码查询
4. 关注召回通知
5. 定期查看召回公告

### Q8：召回通知没收到怎么办？

A：如果未收到召回通知：
1. 主动查询召回信息
2. 联系客服确认
3. 到授权机构检查
4. 关注官方网站公告

### Q9：召回期间可以正常使用车辆吗？

A：根据召回风险等级而定：
- 高风险：建议停止使用
- 中风险：谨慎使用，尽快处理
- 低风险：可正常使用，但建议尽快处理

### Q10：召回需要准备什么材料？

A：召回需要准备：
1. 身份证或驾驶证
2. 行驶证
3. 车辆登记证
4. � 故障报告（如有）
5. 联系方式
