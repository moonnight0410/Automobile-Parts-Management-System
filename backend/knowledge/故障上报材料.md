# 故障上报所需材料

## 概述

为确保故障信息的准确性和完整性，上报故障时需要提供相关材料。完整的材料有助于快速定位问题并制定解决方案。

## 必需材料

### 1. 零部件信息

#### 零部件ID（必需）
- **获取方式**：
  - 扫描零部件二维码
  - 查看零部件包装标签
  - 联系供应商查询
- **格式**：字母数字组合，如：ENG-PISTON-202505
- **重要性**：唯一标识零部件，用于追溯和查询

#### VIN码（必需）
- **定义**：车辆识别代码（Vehicle Identification Number）
- **长度**：17位字符
- **获取位置**：
  - 车辆前挡风玻璃左下角
  - 驾驶员侧车门门框
  - 车辆登记证书
  - 保险单
- **作用**：关联零部件与具体车辆

#### 零部件批次号（必需）
- **来源**：系统自动根据零部件ID填充
- **格式**：BATCH-YYYYMMDD-XXX
- **作用**：追踪同批次其他零部件

### 2. 故障描述

#### 故障现象（必需）
- **要求**：
  - 详细描述故障表现
  - 说明故障发生的环境条件
  - 描述故障的频率和持续时间
- **示例**：
  - "发动机在冷启动时有明显异响，持续约10秒"
  - "刹车踏板踩下时有松软感，制动力不足"
  - "空调制冷效果差，出风口温度偏高"

#### 故障发生时间（必需）
- **格式**：YYYY-MM-DD HH:MM
- **精度**：精确到小时即可
- **作用**：判断故障是否在质保期内

#### 故障发生地点（必需）
- **内容**：城市、区县、具体地点
- **示例**："北京市朝阳区望京街道"
- **作用**：判断是否与地域相关

#### 故障严重程度（必需）
- **可选等级**：
  1. 轻微：不影响正常使用，可稍后处理
  2. 一般：影响部分功能，需尽快处理
  3. 严重：严重影响安全或使用，需立即处理
  4. 危急：存在安全隐患，必须立即停车

### 3. 上传证据

#### 故障照片（必需）
- **数量**：1-5张
- **要求**：
  - 清晰展示故障部位
  - 光线充足，避免模糊
  - 包含零部件整体和局部特写
  - 真实反映故障情况
- **格式**：JPG、PNG
- **大小限制**：每张不超过5MB
- **拍摄建议**：
  - 多角度拍摄（正面、侧面、细节）
  - 使用自然光或补光
  - 避免使用滤镜和美颜
  - 确保背景干净整洁

#### 视频附件（可选）
- **数量**：0-1个
- **要求**：
  - 清晰展示故障过程
  - 声音清晰（如有异响）
  - 时长不超过5分钟
- **格式**：MP4、AVI
- **大小限制**：不超过50MB

#### 其他文档（可选）
- **类型**：
  - 维修记录
  - 检测报告
  - 购买凭证
  - 质保卡
- **格式**：PDF、JPG、PNG
- **作用**：提供更多背景信息

### 4. 联系信息

#### 上报人信息（必需）
- **姓名**：自动填充当前登录用户
- **联系电话**：必填，用于后续联系
- **电子邮箱**：必填，用于接收通知
- **所属机构**：自动显示当前登录机构

## 可选材料

### 1. 维修历史
- **内容**：之前的维修记录
- **作用**：判断是否为重复故障
- **格式**：维修单扫描件或照片

### 2. 使用环境
- **内容**：车辆使用环境描述
- **示例**：
  - 主要在市区行驶
  - 经常在山区道路行驶
  - 气候潮湿多雨
- **作用**：分析故障原因

### 3. 第三方检测报告
- **来源**：授权检测机构
- **作用**：提供专业判断
- **格式**：PDF或扫描件

## 常见问题

### Q1：没有零部件ID怎么办？

A：如果没有零部件ID，可以通过以下方式获取：
1. 联系零部件供应商
2. 查看车辆维修记录
3. 到授权维修中心检测
4. 提供零部件照片和描述，由系统协助识别

### Q2：VIN码输入错误怎么办？

A：VIN码输入错误会导致无法关联车辆。请：
1. 仔细核对17位字符
2. 检查是否有字母O和数字0混淆
3. 确认字母I和数字1的区别
4. 如仍不确定，联系客服协助

### Q3：照片上传失败怎么办？

A：照片上传失败可能的原因：
1. 文件过大：压缩后再上传
2. 格式不支持：转换为JPG或PNG
3. 网络问题：检查网络连接
4. 浏览器问题：更换浏览器或清除缓存

### Q4：可以上传PDF格式的照片吗？

A：不可以。照片必须为JPG或PNG格式。如需上传PDF文档，请使用"其他文档"选项。

### Q5：故障描述写多少字合适？

A：故障描述建议：
- 最少50字，确保信息完整
- 最多500字，避免冗长
- 重点描述：故障现象、发生条件、影响程度

### Q6：视频必须上传吗？

A：视频不是必需的，但强烈建议上传，特别是：
- 故障有声音（如异响）
- 故障有动态表现（如抖动）
- 照片无法清晰展示故障时

### Q7：材料上传后可以修改吗？

A：故障报告提交前可以修改。提交后：
- 一般不能修改已上传的材料
- 如需补充，请联系客服
- 客服会根据情况协助处理

### Q8：如何确保材料真实有效？

A：为确保材料真实有效：
1. 照片和视频应为原始拍摄，不得编辑
2. 文档应为原件扫描件
3. 信息应客观真实，不得虚构
4. 虚假材料将影响质保权益
