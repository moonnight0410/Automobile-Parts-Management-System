# 系统使用指南

## 概述

本系统是基于区块链技术的汽车零部件全生命周期可信管理系统，提供零部件生产、流通、安装、维修、召回等全流程管理。本指南将帮助您快速了解和使用系统功能。

## 系统登录

### 登录方式
1. 打开系统网址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 系统根据用户角色自动分配相应权限

### 忘记密码
1. 点击"忘记密码"链接
2. 输入注册邮箱或手机号
3. 系统发送验证码
4. 输入验证码并重置密码

### 首次登录
1. 首次登录后建议修改密码
2. 完善个人信息
3. 绑定手机号和邮箱
4. 验证身份信息

## 系统导航

### 顶部导航栏
- **首页**：系统首页，显示概览信息
- **零部件管理**：零部件信息查询、生产记录、流通记录
- **售后管理**：故障报告、召回记录、智能售后助手
- **区块链浏览器**：区块链信息、区块查询、交易查询
- **用户中心**：个人信息、密码修改、权限管理

### 侧边导航栏
- 根据用户角色显示不同菜单
- 点击菜单项进入对应功能页面
- 支持多级菜单展开和收起

## 主要功能模块

### 1. 零部件管理

#### 零部件信息查询
- **功能**：查询零部件基本信息、质保信息、流通信息
- **查询方式**：零部件ID、批次号、二维码
- **查询内容**：基本信息、质保信息、流通信息、状态信息

#### 生产记录查询
- **功能**：查询零部件生产记录
- **查询方式**：零部件ID、批次号、生产日期
- **查询内容**：生产时间、生产地点、生产人员、生产数量

#### 流通记录查询
- **功能**：查询零部件流通记录
- **查询方式**：零部件ID、批次号、流通类型
- **查询内容**：入库记录、出库记录、运输记录、安装记录

### 2. 售后管理

#### 故障报告
- **功能**：上报零部件故障，查询处理进度
- **操作流程**：填写故障信息 → 提交报告 → 等待审核 → 查询进度
- **查询方式**：故障ID、零部件ID、VIN码

#### 召回记录
- **功能**：查询召回信息，执行召回处理
- **查询方式**：批次号、零部件ID、VIN码
- **召回流程**：接收通知 → 确认信息 → 预约服务 → 执行召回 → 确认效果

#### 智能售后助手
- **功能**：AI智能问答，提供售后咨询
- **使用方式**：输入问题 → AI识别意图 → 返回答案
- **支持功能**：故障查询、召回查询、质保咨询、进度查询

### 3. 区块链浏览器

#### 区块链信息
- **功能**：查看区块链网络信息
- **显示内容**：区块高度、交易数量、节点数量、网络状态

#### 区块查询
- **功能**：查询指定区块信息
- **查询方式**：区块高度、区块哈希
- **显示内容**：区块哈希、时间戳、交易列表

#### 交易查询
- **功能**：查询指定交易信息
- **查询方式**：交易ID、交易哈希
- **显示内容**：交易ID、交易类型、交易时间、交易内容

### 4. 用户中心

#### 个人信息
- **功能**：查看和修改个人信息
- **修改内容**：姓名、手机号、邮箱、地址

#### 密码修改
- **功能**：修改登录密码
- **操作流程**：输入原密码 → 输入新密码 → 确认新密码 → 提交修改

#### 权限管理
- **功能**：查看用户权限
- **显示内容**：角色权限、功能权限、数据权限

## 常见操作

### 上报故障
1. 进入"售后管理 > 故障报告"页面
2. 点击"新建故障报告"按钮
3. 填写零部件ID、VIN码、故障描述等信息
4. 上传故障照片（1-5张）
5. 点击"提交"按钮
6. 系统生成故障ID并开始处理流程

### 查询故障进度
1. 进入"售后管理 > 故障报告"页面
2. 在搜索框中输入故障ID
3. 点击"查询"按钮
4. 系统显示故障处理进度和状态

### 查询召回信息
1. 进入"售后管理 > 召回记录"页面
2. 在搜索框中输入批次号、零部件ID或VIN码
3. 点击"查询"按钮
4. 系统显示召回信息、召回原因和处理方式

### 查询零部件信息
1. 进入"零部件管理 > 零部件信息"页面
2. 在搜索框中输入零部件ID或批次号
3. 点击"查询"按钮
4. 系统显示零部件详细信息

### 使用AI助手
1. 进入"售后管理 > 智能售后助手"页面
2. 在输入框中输入问题
3. 点击"发送"按钮
4. AI助手识别意图并返回答案

## 常见问题

### Q1：系统支持哪些浏览器？

A：系统支持以下浏览器：
1. Chrome（推荐）
2. Firefox
3. Safari
4. Edge
5. 不支持IE浏览器

### Q2：系统支持移动端访问吗？

A：是的，系统支持移动端访问：
1. 支持手机浏览器访问
2. 支持平板浏览器访问
3. 响应式设计，自动适配屏幕

### Q3：系统数据安全吗？

A：是的，系统数据安全可靠：
1. 所有数据上链存证
2. 采用加密技术保护数据
3. 定期备份数据
4. 符合安全标准和监管要求

### Q4：系统可以导出数据吗？

A：可以。系统支持以下导出方式：
1. 导出为PDF文件
2. 导出为Excel文件
3. 打印数据

### Q5：系统有使用限制吗？

A：系统使用限制：
1. 每次上传照片不超过5张
2. 每张照片不超过5MB
3. 单次查询结果不超过100条
4. 其他限制根据系统配置而定

### Q6：系统支持多语言吗？

A：目前系统仅支持中文，未来将支持多语言。

### Q7：系统有API接口吗？

A：是的，系统提供API接口，供第三方系统集成使用。具体API文档请联系技术支持。

### Q8：系统有使用教程吗？

A：是的，系统提供详细的使用教程：
1. 在线帮助文档
2. 视频教程
3. FAQ常见问题
4. 联系客服获取帮助

### Q9：系统可以离线使用吗？

A：不可以。系统需要在线使用：
1. 需要网络连接
2. 需要访问区块链网络
3. 需要实时查询数据

### Q10：系统有使用时间限制吗？

A：系统使用时间限制：
1. 24小时可访问
2. 系统维护时间除外
3. 维护时间会提前通知
